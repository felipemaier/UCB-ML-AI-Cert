## Overview

The goal of this project is to help a car dealer by predicting the price of a used car based on its features. To do so, 
we analyze a dataset of 426,880 used vehicle listings and use machine learning techniques to build a model that predicts 
vehicle price based on features such as year, mileage, condition, fuel type, transmission, manufacturer, and state.

## Results

After evaluating five machine learning models, we chose a Linear Regression with categorical features as the best model.
This model achieved a Mean Absolute Error of $5,578, meaning that on average the model is able to predict the price of 
a used car within $5,578 of the actual price.
